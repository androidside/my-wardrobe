## Design Specifications

### Photo Upload
- **Size**: Max 400px x 400px
- **Aspect Ratio**: 1:1 (square)
- **Placement**: Centered, not full-width
- **Style**: Border with dashed outline when empty

### Form Fields
- **Width**: Full width within dialog
- **Spacing**: 1.5rem between fields
- **Dropdowns**: Should NOT overlap
  - z-index properly managed
  - Positioned within viewport

### Grid Layout (Gallery)
- **Mobile**: 1 column
- **Tablet**: 2 columns
- **Desktop**: 3-4 columns
- **Gap**: 1.5rem between cards

### Colors
- **Primary**: Indigo/Blue
- **Success**: Green
- **Danger**: Red
- **Neutral**: Gray
